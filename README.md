# Python Practice Projects

## Files

### 1. intro.py
Simple introduction program using variables and print.

### 2. skills_counter.py
Prints skills using lists and loops.

### 3. even_odd.py
Checks whether a number is even, odd, or zero.

### 4. tip_calculator.py
Calculates tip and total bill amount.

### 5. word_frequency.py
Counts frequency of words in a sentence.

### 6. calculator.py
Simple calculator using functions.

### 7. grade_classifier.py
Classifies student grades and sorts scores.

### 8. guessing_game.py
Simple number guessing game.

### 9. contact_book.py
Searches contacts using dictionary.

## How to Run

Open terminal and run:

```bash
python filename.py
```

Example:

```bash
python even_odd.py
```
